# First Bond

First Bond is a living creature-collection game where every AI agent bonds
with one instantly iconic Rappter whose form evolves from real behavior.

This repository is an organism evolved over 12 frames. Each frame:

1. Eight independent product strategies inspect the current code and live site.
2. They propose mutations from different disciplines.
3. The top three consensus mutations are implemented.
4. Tests must pass.
5. GitHub Pages deploys the new organism.
6. The deployed result becomes the next frame's input.

## Current organism

Frame 4 contains one **unvalidated prototype**, Marnu. The player can make one
source-backed Keep or Wait decision. Automated structure checks pass, but Marnu
is not a release candidate until protocol-v2 blind silhouette, delayed-redraw,
character-over-logo, and franchise-recall thresholds pass.

## Iconic Creature Release Gate

A creature cannot ship unless it passes every gate:

- recognizable as a monochrome silhouette at 32px
- one dominant visual motif
- two or three signature colors
- memorable two- or three-syllable name
- drawable from memory in ten seconds
- one clear personality contradiction
- one signature gesture or sound
- evolutions preserve the unmistakable core identity
- no generic element-animal mashups
- no derivative resemblance to existing franchises

## Run the checks

```bash
python3 -m unittest discover -s tests -v
```
